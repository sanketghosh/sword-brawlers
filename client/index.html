Sword Brawlers

Game Control Instructions

If you are creating the game your controls are: 'a' = left, 'd' = right, 'w' = jump and 's' = attack

If you are not creating the game your controls are: 'left arrow' = left, 'right arrow' = right, 'top arrow' = jump and 'down/bottom arrow' = attack

Please wait until your opponent joins.
